# 发布文档

### 功能特性
- ✨ 基础图片生成功能
  - 支持多种生成风格（写实、卡通、艺术）
  - 实时预览生成结果
  - 图片下载功能

### 技术栈
- 前端：HTML/CSS/JavaScript
- 后端：Node.js + Express
- API：DeepSeek API
- 部署：Nginx
