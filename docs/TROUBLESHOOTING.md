# 故障排除指南

## 常见问题

### 1. 环境变量问题
#### 症状
- 部署失败，提示环境变量或 secret 不存在
- 环境变量无法正确读取

#### 解决方案
1. 检查语法
   - 确保 vercel.json 使用 ${} 语法
   - 不要混用 @secret 语法

2. 验证配置
   - 使用 vercel env ls 检查环境变量
   - 确认变量名称完全匹配

### 2. 部署失败
#### 症状
- 构建失败
- 部署超时
- 环境配置错误

#### 解决方案
1. 检查项目关联
   ```bash
   vercel link
   ```

2. 验证配置文件
   - 检查 vercel.json 语法
   - 确认构建命令正确

3. 环境变量设置
   - 重新配置环境变量
   - 验证变量值正确

## 最佳实践
1. 始终使用一种部署方式
2. 遵循统一的配置语法
3. 保持文档更新
